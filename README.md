# نظام حساب التكاليف المعمارية

نظام متكامل لحساب تكاليف المنشآت المعمارية وجداول الكميات (BOQ) مبني بـ Next.js.

## المميزات

- **حاسبة التكاليف التفاعلية**: حساب تكاليف 11 نوع من الأعمال الإنشائية
  - أعمال الحفر والردم
  - خرسانة القواعد والأعمدة والسقوف
  - أسقف الهوردي
  - أعمال البياض والدهانات والبلاط
  - أعمال البلوك والعزل

- **إدارة المشاريع**: إنشاء وتتبع المشاريع مع المراحل والبنود
- **أسعار المواد**: إدارة قائمة أسعار المواد الإنشائية
- **التقارير**: تصدير تقارير التكاليف (TXT, CSV)
- **دعم متعدد العملات**: ريال يمني، ريال سعودي، دولار أمريكي، درهم إماراتي
- **واجهة عربية كاملة**: دعم RTL بالكامل

## التقنيات المستخدمة

- [Next.js 16](https://nextjs.org/) - إطار عمل React
- [TypeScript](https://www.typescriptlang.org/) - لغة البرمجة
- [Tailwind CSS 4](https://tailwindcss.com/) - تصميم CSS

## التشغيل

```bash
npm install
npm run dev
```

افتح [http://localhost:3000](http://localhost:3000) في المتصفح.

## البناء

```bash
npm run build
npm start
```

## هيكل المشروع

```
src/
├── app/
│   ├── calculator/     # حاسبة التكاليف
│   ├── projects/       # إدارة المشاريع
│   ├── materials/      # أسعار المواد
│   ├── reports/        # التقارير
│   ├── layout.tsx      # التخطيط الرئيسي
│   └── page.tsx        # لوحة التحكم
├── components/
│   ├── Sidebar.tsx     # الشريط الجانبي
│   └── ProjectDetailsView.tsx  # تفاصيل المشروع
└── lib/
    ├── types.ts        # التعريفات
    ├── constants.ts    # الثوابت والمواد الافتراضية
    ├── calculations.ts # محرك الحسابات
    └── storage.ts      # تخزين البيانات
```

## محرك الحسابات

يطبق النظام المعايير الهندسية في حصر الكميات:

| نوع العمل | المعامل |
|---|---|
| أعمال الحفر | معامل انتفاش 1.25 |
| أعمال الردم | معامل دمك 1.20 |
| خرسانة القواعد | حديد 90 كجم/م3 |
| خرسانة الأعمدة | حديد 130 كجم/م3 |
| خرسانة السقوف | حديد 110 كجم/م3 |
| أسقف هوردي | حديد 80 كجم/م3 + بلوك 12.5 قطعة/م2 |
